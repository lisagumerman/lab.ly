Lab.ly

sexy big data science for real people

Overview

Lab.ly is a cloud-based platform for crunching your big data and making beautiful visualizations.

Want to get more utility out of your Google Analytics? Cool, we got you. Want to clean up that .csv file and chart column 1 vs column 2 and then drop it into your slideshow? No problem. Need to scrape a webpage, do some nifty logistical regression, and save all that to your local machine? You know what the answer is.

With Lab.ly, YOU are the data scientist.

How it works

  1. Open your Lab.ly workspace
  2. Add a data source -- a .csv file, a url, a Google Analytics account, etc. We'll process it and hand it back to you
  3. Select the column you want to perform wicked sick visualizations on (anything from machine learning to a pretty line graph)
  4. Check out the resulting visualization
  5. Redo the visualization, save it out as an image, create a second visualization, or save the whole workspace and call it a day

Roadmap

In the long run we plan on having an on-prem offering; hookups to Google Analytics, Bing Analytics, SQL data sources (Vertica, PostgreSQL, SQL Server); and a whole bunch more. Here's what's coming:

Beta

Our free version!

  • Upload a file <= 500 million rows
  • Graph time series or histograms
  • Save data and visualizations to your local machine

In the near future

  • Hook up Google Analytics
  • Graph scatter plots, linear regression and polynomial regression visualizations
  • Get summarizations and rollups of your data
  • Scrape a webpage
  • Have an account with multiple associated workspaces, data sources and users
  • Save workspaces for later use

In the faintly more distant future

  • Theme your workspaces with your company colors
  • Automate the population of new data into your account
  • Turn your data source and the analytics on it into a web service to use elsewhere
  • Graph K-mean, K-nearest-neighbors, logistic regression, or support vector machine visualizations
  • Host an on-premise Lab.ly instance
  • Hook up SQL data sources
